## 技术架构

```
前端：微信原生小程序 + Vant Weapp 组件库
后端：Java SpringBoot（推荐）
数据库：MySQL + Redis
AI：垂类大模型 + 亳州药材数据集
图像：AI舌象识别模型（本地轻量部署）
存储：腾讯云COS
支付：微信小程序支付
推送：微信订阅消息
```

## API接口基础信息
- 前缀：`https://api.qichenkj.com/health/api/v1`
- 认证：Bearer Token（微信登录自动获取）
- 格式：JSON，统一返回 `{code, msg, data}`

## 游戏化设计亮点（00后专属）
1. **段位系统**：青铜行者→白银药师→黄金华佗门徒→铂金养生达人→王者秘境宗师
2. **药材碎片合成**：收集碎片→合成卡牌→兑换奖励的收集闭环
3. **体质分身PK**：基于养生习惯数据的好友对战系统
4. **每日盲盒**：完成打卡任务开盲盒，随机获得虚拟道具/优惠券
5. **消消乐**：药材元素三消游戏，积分直接抵扣花茶购买金额
6. **剧情游戏**：华佗古风文字冒险，选择养生习惯决定结局
